Local Storage

Web Storage

  • Very simple api
  • Supported on all modern browsers for several years
  • Unstructured data
  • Poor performance for large/complex data

Web SQL database

  • Good performance
  • Supports transactional database model
  • Supported on mobile, chrome, safari, and opera
  • Deprecated. Not supported on IE or firefox, probably will be removed from other browsers

Indexed Database

  • Good performance
  • supports transactional database model and versioning
  • Very complex API

File System

  • Can store large files with good performance (use responsibly!)
  • No transaction or indexing support
