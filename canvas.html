Canvas and SVG

  • Both allow you to make interactive animations without using flash
  • Javascript can modify the canvas elements and svg elements
  • SVG is based on xml, and elements are remembered after drawing, allowing you to detormine which element was clicked
  • Changes to svg can be as simple as changing element coordinates
  • In canvas, elements are forgotten after being drawn, and updates redraw the entire element
